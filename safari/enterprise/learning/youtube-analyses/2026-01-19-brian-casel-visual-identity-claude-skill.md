# Replacing my n8n workflow with a Claude Code Skill

## TL;DR

Brian Casel demonstrates how to build a comprehensive visual identity system using Claude as a thought partner, then replace an n8n automation workflow with a Claude Code skill for generating on-brand illustrations. The video provides a framework for mapping content themes to visual approaches and establishing consistent brand elements.

## Key Takeaways

### 1. Visual Identity System Structure

Build a documented visual identity system with:
- **Core Philosophy** - Guiding principles (confident minimalism, imperfect on purpose, white space as content, warmth through restraint)
- **Recurring Elements (Constants)** - Objects that appear consistently to build brand recognition (the dog, the mug, the notebook, locations)
- **Style Options** - Defined visual styles with specific characteristics (ink & wash, risograph minimal, woodcut modern, etc.)
- **Content-to-Visual Mapping** - Framework for connecting content themes to illustration approaches

### 2. Claude as Design Thought Partner

Use Claude for collaborative design decisions:
- Paste inspiration images (from Dribbble, Pinterest) into Claude
- Ask Claude to analyze and describe visual styles in precise language
- Iterate on descriptions to refine the style articulation
- Use refined descriptions as prompts for image generation tools

### 3. Illustration-to-Content Mapping Framework

Four approaches for matching visuals to content:
- **Direct** - Literal representation (article about notebooks shows notebook)
- **Metaphorical** - Abstract concepts need visual metaphors (iteration = woodworking plane)
- **Atmospheric** - Setting mood (deep work = late-night desk with lamp)
- **Emotional** - Conveying feeling (frustration = crumpled paper or concerned dog)

### 4. From n8n Automation to Claude Code Skill

n8n workflow limitations:
- Required separate Slack interface for triggering
- Complex multi-node automation setup
- Hard to iterate and modify

Claude Code skill advantages:
- Integrated into existing Claude workflow
- Natural language interface
- Easy to modify and extend
- Maintains full context of brand guidelines
