# First Principles Analysis

Deconstruct a problem to its fundamental truths and build up from there.

## Steps

1. **Identify the problem**
   - What are we trying to solve?
   - What is the current state?
   - What is the desired state?

2. **Question assumptions**
   - List all assumptions being made
   - Challenge each: "Why do we believe this?"
   - Identify which are facts vs opinions

3. **Break down to fundamentals**
   - What are the core components?
   - What are the basic requirements?
   - What constraints are truly immovable?

4. **Rebuild from basics**
   - Start with only proven facts
   - Build up the simplest solution
   - Add complexity only when necessary

5. **Validate approach**
   - Does this solve the original problem?
   - Is it simpler than the previous approach?
   - What are the trade-offs?

## Output
- Problem breakdown
- Identified assumptions (valid/invalid)
- Proposed solution built from first principles
- Trade-off analysis

## Notes
- Use for complex problems where existing solutions feel wrong
- Helps escape "we've always done it this way" thinking
