1. Arrow Components

Directional arrows with arrowhead markers. Includes static and animated variants.

Dark Background
Light Background
Copy this SVG snippet:
<!-- Arrow with marker definition --> <svg width="200" height="50"> <defs> <marker id="arrowhead" markerWidth="10" markerHeight="7" refX="9" refY="3.5" orient="auto"> <polygon points="0 0, 10 3.5, 0 7" fill="#9B1C1C"/> </marker> </defs> <line x1="20" y1="25" x2="180" y2="25" stroke="#9B1C1C" stroke-width="2" marker-end="url(#arrowhead)"/> </svg> <!-- Animated arrow (add inside <line>) --> <line ... stroke-dasharray="5,5"> <animate attributeName="stroke-dashoffset" from="10" to="0" dur="0.5s" repeatCount="indefinite"/> </line>
