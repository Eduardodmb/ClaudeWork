# Safari Brand Guide

## Purpose

This guide establishes the visual identity standards for Safari Circuits and Safari Trace products. Consistent application of these guidelines ensures brand recognition and professional presentation.

## Logo Usage

### Primary Logo

The Safari logo features a diamond-shaped "S" icon with circuit board patterns.

```
Logo Files:
├── safari-logo.svg              # Full logo (dark backgrounds)
├── safari-logo-white.svg        # Full logo (light backgrounds)
├── safari-icon-diamond-s.svg    # Icon only (dark backgrounds)
└── safari-icon-diamond-s-red.svg # Icon only (red version)
```

### Logo Clear Space

Maintain minimum clear space around the logo equal to the height of the "S" icon:

```
        ┌─────────────────────────────────────────┐
        │                                         │
        │    X  ┌─────────┐                       │
        │   ◄──►│  ◇ S    │ SAFARI CIRCUITS       │
        │       └─────────┘                       │
        │    ▲                                    │
        │    │X                                   │
        │    ▼                                    │
        └─────────────────────────────────────────┘

        X = Minimum clear space (height of icon)
```

### Minimum Size

| Usage | Minimum Width |
|-------|---------------|
| Print | 1 inch / 25mm |
| Digital | 120 pixels |
| Icon only | 32 pixels |
| Favicon | 16 pixels |

### Logo Don'ts

| Don't | Reason |
|-------|--------|
| Stretch or distort | Maintains proportions |
| Change colors | Brand consistency |
| Add effects (shadows, etc.) | Clean presentation |
| Place on busy backgrounds | Legibility |
| Rotate the logo | Professional appearance |
| Modify icon design | Brand integrity |

## Co-Branding

### Partner Logos

When displaying with partner logos:

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  ◇ Safari Circuits    |    Partner Logo    |    Partner 2   │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Rules:
- Equal visual weight
- Consistent spacing
- Divider between logos
- Safari logo on left (primary position)
```

### Customer Logos

Safari logo should appear with equal or greater prominence than customer logos in joint materials.

## Voice and Tone

### Brand Voice

| Attribute | Description | Example |
|-----------|-------------|---------|
| Professional | Industry expertise | "Leveraging decades of EMS experience" |
| Clear | Easy to understand | "Track every component in real-time" |
| Confident | Assured but not arrogant | "Proven traceability solutions" |
| Helpful | User-focused | "Let us help you achieve compliance" |

### Writing Guidelines

```
Do:
- Use active voice
- Be concise and direct
- Focus on benefits
- Use industry terminology appropriately

Don't:
- Use jargon without explanation
- Make unsubstantiated claims
- Use overly casual language
- Use excessive exclamation points
```

### Product Names

| Product | Correct Usage | Avoid |
|---------|--------------|-------|
| Safari Trace | "Safari Trace" or "Trace" | "SafariTrace", "ST" |
| Safari Assist | "Safari Assist" or "Assist" | "SafariAssist", "SA" |
| Safari Analyze | "Safari Analyze" or "Analyze" | "SafariAnalyze" |
| Safari Studio | "Safari Studio" or "Studio" | "SafariStudio", "SS" |
